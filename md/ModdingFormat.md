# Modding Format

To create compaitble mods for this launcher you most follow this format

Required:
Folder (you can call this the mods name)
- Mod-Windows.pak
- Mod-Windows.utoc
- Mod-Windows.ucas
Optional:
- name.txt (this will be the mods name in the launcher)
- description.txt (this will be the mods description in the launcher)
- splash.png (this will be the mods cover/splash image in the launcher)
